Data integration consists of two basic steps:

  • semantic enrichement
  • mapping discovery
LAV vs. GAV

LAV and GAV describe two approaches for integrating multiple ontologies.

  • GAV (=global-as-view) => the ontologies are combined in a global view
    • each concept of the global view is mapped to a query over the sources
    • global is responsible for querying (and translating the sources)

  • LAV (=local-as-view) => sources are views over the global view
    • local is responsible for translation to global
