The January issue covers the following interesting topics:

Tag Clouds Jurgen Appelo provides hints for optimizing the presentation of tag clouds by linearizing the data according to a Pareto distribution. This technique could be useful for assigning the weight of labels in our ontologies.

Google Web Toolkit Ed Burnette and Adam Houghton present an example application using the Google Web Toolkit (GWT) to provide a Flickr image service.

Creating a Web 2.0 application with PHP Andi Gutmans and Cal Evans present a graphical news feed service, extracting key words from news feeds and presenting flickr graphics related to the extracted terms. The Yahoo Content Analyzer service is facilitated for determining the relevant keywords.
