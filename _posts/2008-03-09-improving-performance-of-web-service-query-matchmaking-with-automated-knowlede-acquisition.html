by Gupta et al.

The Article presents a systems for simple interfacing Web services using an HTML based search engine, mapping the queries to Web service requests.

A Web service is an interface to application functionality accessible using well-known Internet standards independent of any operating system or programming language

Based on an ontology and wordnet's synonym detection, the system tries mapping user requests to the appropriate Web service calls. Memorized optimization facilitates data from previous queries for the benefit of future requests.
